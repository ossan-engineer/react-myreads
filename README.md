# React My Read

This is a virtual bookcase to store your books and track what you're reading. You can search for books and add them to a bookshelf.

## Demo

http://react-myreads.s3-website-ap-northeast-1.amazonaws.com

## Getting Started

```
$ yarn
$ yarn start
```
